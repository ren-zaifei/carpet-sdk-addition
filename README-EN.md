# Carpet SDK Addition

[![License](https://img.shields.io/badge/license-LGPLv3-blue.svg)](https://opensource.org/licenses/LGPL-3.0)
[![Fabric](https://img.shields.io/badge/Fabric-supported-orange.svg)](https://fabricmc.net)

A Carpet mod extension tailored for the **SDK Minecraft server**, adding a variety of interesting and practical Carpet rules to help server administrators and players optimize and enhance their gameplay experience.

## Overview

Carpet SDK Addition is a Fabric-based extension for the Carpet mod. It builds upon the original Carpet mod by introducing a series of practical rules, making server operation more efficient and gameplay more diverse and enjoyable.

> **Note**: This mod must be used together with the original Carpet mod (it indirectly depends on Fabric API).

## Features

- Flippin Totem  Use totem of undying to change block's facing.

For details on available rules, please check the source code or use the `/carpet` command in-game to query them.

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/installer/).
2. Download and install [Fabric API](https://modrinth.com/mod/fabric-api).
3. Download and install the latest version of the [Carpet mod](https://www.curseforge.com/minecraft/mc-mods/carpet).
4. Place the `carpet-sdk-addition-x.x.x.jar` file into the `mods` folder of your server or client.
5. Launch the game/server to start using it.
