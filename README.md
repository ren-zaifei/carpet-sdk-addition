English version [docs](https://github.com/ren-zaifei/carpet-sdk-addition/README-EN.md)

一个为 **SDK Minecraft 服务器** 量身定制的 Carpet 模组扩展，增加了多个有趣且实用的 Carpet 规则，帮助服务器管理者和玩家更好地优化游戏体验。

## 简介

Carpet SDK Addition 是基于 Fabric 的 Carpet 模组扩展。它在原版 Carpet 模组的基础上添加了一系列实用规则，让服务器运行更加高效、玩法更加丰富。

> **注意**：本模组必须与 Carpet 模组一起使用（通过 Fabric API 间接依赖）。

## 特性
- 不死图腾扳手 使用不死图腾来改变方块朝向
- 更好的猪灵AI 猪灵的首格物品为金胡萝卜时，则只会保留交易AI

更多规则详情请查看源码或游戏内使用 `/carpet` 命令查询。

## 安装步骤

1. 安装 [Fabric Loader](https://fabricmc.net/use/installer/)。
2. 下载并安装 [Fabric API](https://modrinth.com/mod/fabric-api)。
3. 下载并安装最新版的 [Carpet 模组](https://www.curseforge.com/minecraft/mc-mods/carpet)。
4. 将 `carpet-sdk-addition-x.x.x.jar` 放入服务器/客户端的 `mods` 文件夹。
5. 启动游戏/服务器即可使用。
